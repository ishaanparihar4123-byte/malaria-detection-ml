# 🦠 Malaria Detection using Deep Learning

A deep learning based web application that detects malaria infection from microscopic blood cell images.

The system uses a Convolutional Neural Network (CNN) trained on malaria cell images to classify whether a cell is **Parasitized** or **Uninfected**.

---

# 🚀 Features

* Image upload interface
* Deep learning based classification
* Flask web application
* TensorFlow/Keras model
* Real-time prediction

---

# 🧠 Model

The model is a CNN trained on malaria cell images to detect infected red blood cells.

Prediction classes:

* **Parasitized**
* **Uninfected**

# ⚙️ Installation

Clone the repository:

```
git clone https://github.com/ishaanparihar4123-byte/malaria-detection-ml.git
cd malaria-detection-ml
```

Install dependencies:

```
pip install -r requirements.txt
```

Run the application:

```
python app.py
```

---

# 📸 Usage

1. Open the web interface
2. Upload a blood cell image
3. The system predicts if the cell is infected or not
